# Lab Logos

Please save the lab logo images in this directory with the following names:

1. **melady.png** - Melady Lab logo (black text "Melady Lab")
2. **ink.png** - INK Lab logo (brain with colorful dots and "ink" text)
3. **dill.png** - DILL Lab logo (green dill herb with "DILL" text)
4. **glamor.png** - GLAMOR Lab logo (yellow/gold stars with "GLAMOR" text)
5. **lime.png** - LIME Lab logo (green lime slice with happy face and "LIME FEELIN' GOOD" text)

The component is configured to load these images from:
`/images/labs/{lab-name}.png`
